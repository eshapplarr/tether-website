# Tether

**A foster-first platform connecting dogs with homes that fit them.**

Tether is a matching system for Houston-area shelters and the fosters who can help them. Instead of fosters scrolling through endless lists or shelters posting to Facebook in hope, Tether matches dogs to homes based on what actually fits: lifestyle, household, experience, and the dog's real temperament and needs.

This repository holds the source code for tether.dog — the public website and the shelter dashboard.

## Status

In development. v0.1 spec finalized June 2026. Pre-launch.

## What's planned for v0.1

- A homepage explaining Tether's mission
- Four entry points: shelters, experienced fosters, new fosters, people who've found a stray dog
- A shelter dashboard for managing dogs and viewing matches
- A matching engine with hard filters and weighted soft scoring
- A broadcast system for urgent placements

## Why this exists

Houston has one of the highest dog euthanasia rates in the United States. Most of those dogs don't die from neglect or because people don't care — they die from a lack of connection. Tether is built to fix that hole in Houston's shelters.
